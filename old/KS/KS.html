bot作成のサンプルプログラムです． 次のファイルをダウンロードして下さい． tweet.py

使い方
まずは必要なパッケージのインストールから．
端末上で次のコマンドを順に実行します． 

wget http://peak.telecommunity.com/dist/ez_setup.py
sudo python ez_setup.py
sudo easy_install pip
sudo easy_install tweepy
これができたら，実際にサンプルプログラムを実行してみましょう． 次のコマンドで実行できます．(予めtweet.pyをダウンロードしたディレクトリに移動しておいて下さい) 

python tweet.py
実行すると文字列の入力を求められるので，テキトーな文字列を入力して下さい． すると，サンプル用のアカウント@KeisanSugaku15でその内容が呟かれます．
pythonの知識が多少あれば，サンプルプログラムを弄ることで，pythonプログラムが生成した文字列を呟かせることも簡単にできるでしょう．
(当然ツイート内容は全世界に向けて発信されているので，良識を持って利用してください)
